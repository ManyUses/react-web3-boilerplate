# react-web3-boilerplate

A boilerplate react template forked from Uniswap.

* Nextjs
* Typescript
* Tailwind
* Lingui i18n
* web3-react: All EVM the wallets and chains
* Services: 
    * Covalent
    * The graph
    * Generic Web3
* Lottie animations
* Hooks galore
* ENS Tools
* Token/Contract tools
